# MPP Compliance Doctor MCP

Production-ready MCP server that:

- Scans MCP/API servers
- Explains MPP Scan warnings
- Generates fixes automatically

## Tools

- debug_mpp_scan_results
- scan_openapi_spec
- validate_402_response
- generate_openapi_payment_patch
- generate_nextjs_402_fix
- generate_express_402_fix
- explain_mpp_warning

## Run

```bash
npm install
npm run dev
